# Extension Repositories

Due to DMCA takedown notices, **Sugoi** does not come with preloaded extensions. To browse and read manga, you have to sideload third-party extension repositories.

::: danger Caution
Beware that third-party repositories or extensions have full access to the app and could potentially contain malware. Install them at your own risk.
:::

## How-to

::::tabs
== Adding Repository
* **Auto:** Click on **Install** in the table above from your Android device. Sugoi will open and ask to add the repo automatically.
* **Manual:** Copy the **Manual Repository JSON URL** above. In Sugoi, go to **More** -> **Settings** -> **Browse** -> **Extension repos** -> **Add repository**, and paste the URL.

== Adding Extension
1. After adding a repository, go to **Browse** -> **Extensions** tab.
2. Find your desired source in the list and tap **Install**.
3. Once installed, if prompted by Android, allow installing unknown apps from Sugoi.
4. Extensions may need to be trusted/enabled by clicking the shield check icon before they can be used.
::::
